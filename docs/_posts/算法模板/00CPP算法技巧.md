## 读 char 使用 %s

为了防止题目给出的数据结尾多一个空格或者回车
那么正确的逻辑代码页不能 AC，用 %s 读字符就不会存在这个问题，%s 会自动跳过

``` cpp
char op[2];
int a, b;
scanf("%s%d%d", op, &a, &b);
if (*op == 'M')
p[find(a)] = find(b);
```
